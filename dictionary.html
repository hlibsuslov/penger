Dictionary

All 2048 words of the BIP39 standard, each mapped to a unique 12-bit index (1–2048). The index determines the punch pattern (top → bottom) on a metal backup plate — filled = punch hole, empty = skip. Click any word to see the full breakdown.

You can interact with the full BIP39 dictionary offline using the PDF version.

How to read an index
Example: 0985 = inner

Index 0985 in binary code: 1 is strike, 0 is skip. The order from left to right (MSB → LSB) corresponds to filling the column from top to bottom.
