[ HOW IT WORKS ]

Binary Encoding Logic

BIP39 defines 2048 words indexed 0–2047. Since 2048 = 211, each index fits in exactly 11 binary digits. Each word count uses different entropy lengths with a SHA-256 checksum appended.

12 words128-bit entropy + 4-bit checksum
15 words160-bit entropy + 5-bit checksum
18 words192-bit entropy + 6-bit checksum
21 words224-bit entropy + 7-bit checksum
24 words256-bit entropy + 8-bit checksum
