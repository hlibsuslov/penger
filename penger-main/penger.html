[ INSTRUCTION ]

Как записать seed на титановую пластину How to encode your seed onto titanium

Каждое слово BIP39 имеет порядковый номер (индекс) от 0 до 2047. Этот номер переводится в 11-битный двоичный код. Вы набиваете код кернером на пластину: удар = 1, пропуск = 0. Ниже — пошаговый процесс. Every BIP39 word has an index number from 0 to 2047. This number converts to an 11-bit binary code. You punch the code onto the plate: punch = 1, skip = 0. Here's the step-by-step process.

01

Найдите индекс каждого слова Find the index of each word

Откройте список BIP39 (2048 слов). Каждому слову соответствует номер. Например: Open the BIP39 wordlist (2048 words). Each word has a number. For example:

СловоWord ИндексIndex Двоичный кодBinary
abandon 0 00000000000
ability 1 00000000001
abstract 6 00000000110
crisp 427 00110101011
ocean 1199 10010101111
zoo 2047 11111111111
02

Переведите индекс в двоичный код (11 бит) Convert the index to binary (11 bits)

Каждый индекс записывается ровно 11 цифрами (нулями и единицами). Число 2048 = 211, поэтому любое слово из списка вмещается в 11 бит. Пример: Each index is written as exactly 11 digits (zeros and ones). Since 2048 = 211, any word from the list fits into 11 bits. Example:

СловоWord: ocean ИндексIndex: 1199 ДвоичныйBinary: 1 0 0 1 0 1 0 1 1 1 1 ПозицияPosition: 1024 512 256 128 64 32 16 8 4 2 1
03

Набейте код кернером на пластину Punch the code onto the plate

На пластине 12 строк (по одной на слово) и 11 позиций в каждой строке. Для каждого бита: 1 = удар кернером (точка), 0 = пропуск (пусто). Пример — seed из 4 слов: The plate has 12 rows (one per word) and 11 positions per row. For each bit: 1 = punch (dot), 0 = skip (empty). Example — 4-word seed:

04

Проверьте запись Verify the record

Пройдите по каждой строке: переведите набитый двоичный код обратно в число, найдите слово по индексу. Совпадает с оригиналом — запись верна. В наборе есть тренировочный шаблон для отработки до финальной записи. Go through each row: convert the punched binary back to a number, find the word by index. If it matches the original — the record is correct. The kit includes a training template to practice before the final recording.

05

Поместите пластину в безопасное физическое место. Титан не боится воды, огня и коррозии. Seed записан в формате, который не читается без знания системы. Это ваш финальный backup. Place the plate in a secure physical location. Titanium resists water, fire, and corrosion. The seed is recorded in a format that can't be read without knowledge of the system. This is your final backup.
