[ BIP39 / BINARY ENCODER ]

Seed Phrase Generator

Generate a cryptographically random BIP39 seed phrase with proper SHA-256 checksum. Each word index (1–2048) is encoded as a 12-bit binary value and visualized as a dot matrix for metal backup encoding.

Example:

[ BIP39 / BINARY DECODER ]

Binary Decoder

Input binary patterns by clicking dots on the grid. Each column decodes into a BIP39 word in real time. Valid indices are 1–2048; all zeros (000000000000) and any value >2048 are marked invalid.

Example:

0001 = Abandon

Click the dots in each column to toggle bits. The column header shows the live computed index. Row labels show bit weight and bit number.
